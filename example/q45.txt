45. Difference between object-oriented and object based language.

a)
-->object-oriented supports all the features of the oops.
-->object based language does not support all feature of oops.

b)
-->object-oriented supports inheritance.
-->object based language not support oops concepts inheritance.

c)
-->c#,java,vb,net are the example of the object-oriented.
-->javascript,VB are examples of the object based language.
